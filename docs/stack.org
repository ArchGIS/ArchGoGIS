Если будем переписывать с Go на другой язык, каковы критерии выбора:
1) удобная работа с JSON (желательно внутри std)
2) наличие хороших neo4j+postgresql драйверов
3) хорошие network библиотеки (желательно внутри std)
4) шаблоны/генерики или встроенные функции для работы с коллекциями (find, filter, map, ...)
5) наличие хорошей документации по языку и его std
6) хорошая работа под Linux
7) без significant space (то есть Python - точно нет)
8) наличие исключений или продолжений (continuations)
